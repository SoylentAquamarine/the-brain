System Overview — The Brain

Blueprint · Last updated 2026-05-02

Purpose

The Brain is a Python CLI orchestration system that offloads AI tasks from Claude to free provider APIs. Claude acts as orchestrator — deciding what to do and which worker to use — while the actual AI work (summarization, code generation, classification, etc.) is handled by free-tier providers. All calls are logged for cost tracking.

Core promise: Claude only does what only Claude can do. Everything else is delegated.

CLI Interface

python delegate.py --provider <name> --type <type> --prompt "<prompt>"

Claude reads the routing table in CLAUDE.md and calls delegate.py directly. The result is returned on stdout and relayed to the user.

Integration with Claude Code

The Brain is activated by adding a routing table to the global CLAUDE.md (at C:\Users\<user>\.claude\CLAUDE.md). Claude reads this at session start and automatically delegates matching tasks rather than answering them directly.

Session startup runs session_start.py which prints live provider health and usage stats so routing advisories can be relayed before any work begins.
